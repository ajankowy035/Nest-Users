<p>The simple app created in purpose of learning and nest mastering :)</p>

## Description

before running specify .evn.development and .env.test files with NODE_ENV variable it's used by app to set up the enviroment.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
